API samples

Course List

localhost:8000/api/course/list

Course detail with lessons sorted by data

localhost:8000/api/course/detail/(course_id:int)

Register new user (POST only)

localhost:8000/api/user/register 
Sample:
{
  "full_name": "Aleksey",
  "email": "aleksey@mail.ru",
  "password": "password"
}

Login user (POST only)

localhost:8000/api/user/login 
Sample:
{
  "email": "aleksey@mail.ru",
  "password": "password"
}

Students on course (GET, only for registered users)

localhost:8000/api/course/detail/(course_id:int)/students

Add student on course (POST, only for registered users)

localhost:8000/api/course/detail/(course_id:int)/students
